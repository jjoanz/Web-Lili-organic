OPCIÓN 1: COMANDOS MANUALES (RECOMENDADO) ⌨️
═══════════════════════════════════════════════════════════════════════

Abre Visual Studio Code y la terminal (Ctrl + `), luego:

# 1. Navegar a tu carpeta de proyectos
cd Documents

# 2. Clonar el repositorio
git clone https://github.com/jjoanz/Web-Lili-organic.git

# 3. Entrar a la carpeta
cd Web-Lili-organic

# 4. Copiar todos los archivos descargados a esta carpeta

# 5. Agregar todo
git add .

# 6. Commit
git commit -m "✨ Proyecto inicial Lili Organic"

# 7. Push
git push origin main

¡LISTO! ✅


OPCIÓN 2: SCRIPT AUTOMÁTICO (Mac/Linux) 🚀
═══════════════════════════════════════════════════════════════════════

1. Clonar el repo:
   git clone https://github.com/jjoanz/Web-Lili-organic.git

2. Copiar todos los archivos del proyecto a Web-Lili-organic/

3. En la terminal, dentro de la carpeta Web-Lili-organic:
   bash subir-a-github.sh

¡LISTO! ✅


OPCIÓN 3: SCRIPT AUTOMÁTICO (Windows) 🪟
═══════════════════════════════════════════════════════════════════════

1. Clonar el repo:
   git clone https://github.com/jjoanz/Web-Lili-organic.git

2. Copiar todos los archivos del proyecto a Web-Lili-organic\

3. Doble click en:
   subir-a-github.bat

¡LISTO! ✅

═══════════════════════════════════════════════════════════════════════

Después de subir el proyecto:

1. Ir a: https://github.com/jjoanz/Web-Lili-organic
2. Click en "Settings"
3. En el menú izquierdo: "Pages"
4. En "Source", seleccionar: "main" branch
5. Click "Save"
6. ¡Esperar 2-3 minutos!

Tu web estará en vivo en:
👉 https://jjoanz.github.io/Web-Lili-organic/

❓ SI TIENES PROBLEMAS:
═══════════════════════════════════════════════════════════════════════

PROBLEMA: "git: command not found"
→ Instalar Git: https://git-scm.com/downloads

PROBLEMA: "Authentication failed"
→ Usar token de GitHub en lugar de contraseña
→ Crear token en: GitHub → Settings → Developer settings

PROBLEMA: "Permission denied"
→ Verificar que tienes acceso al repositorio
